Daemonkey
一个记住你所想，与你一起成长，有七十二变的 AI 搭档

先给你的 Daemonkey 一把钥匙

它跑在你自己的 LLM key 上（你的钥匙只存在本机 .env，不上传任何地方）。

没有 key？以 DeepSeek 为例（3 步搞定）：
  1. 浏览器打开 platform.deepseek.com，注册并登录。
  2. 左侧「API keys」→「创建 API key」→ 复制 sk- 开头那一长串（只显示一次，记得存好）。
  3. 回到这里：服务商选「DeepSeek 官方」，把 key 粘进下面的「API Key」框，点保存即可。

接口地址 base_url 是服务商的 API 入口——选了预设会自动填好（DeepSeek 是 https://api.deepseek.com），一般不用动。换别的服务商（如 GLM 智谱）就选对应预设，或选「自定义」填它给你的地址。模型 model 已替你选好默认（DeepSeek 是 deepseek-v4-pro），一般不用动。

默认已给你选好——填了 DeepSeek 的 key 就直接用上目前最好的 deepseek-v4-pro。想换可改：GLM 常用 glm-4.6。
